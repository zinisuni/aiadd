## 다이어그램 작성 도구
다이어그램은 Mermaid를 사용하여 작성되었습니다. Mermaid는 텍스트 기반의 다이어그램 도구로, 마크다운 파일에서 직접 다이어그램을 작성하고 렌더링할 수 있습니다.

### Mermaid 특징
- 텍스트 기반으로 버전 관리가 용이
- 마크다운과 통합이 쉬움
- 다양한 종류의 다이어그램 지원
- GitHub에서 기본적으로 렌더링 지원
